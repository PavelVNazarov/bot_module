Для развертывания Вашего Django проекта на сервере и настройки токена Telegram бота, выполните следующие шаги:

1. Подключение к серверу по SSH
Откройте терминал и выполните команду:

ssh root@5.35.89.79
Введите пароль ???? при запросе.

2. Установка необходимых зависимостей
Убедитесь, что на сервере установлены Python и pip. Если они не установлены, выполните:

apt update
apt install python3 python3-pip

3. Клонирование Вашего проекта
Если Ваш проект находится в репозитории, клонируйте его:

git clone <URL_вашего_репозитория>
cd <имя_папки_проекта>

4. Установка зависимостей проекта
Установите зависимости, указанные в requirements.txt:

pip install -r requirements.txt

5. Настройка токена бота
Откройте файл настроек Вашего Django проекта (обычно это settings.py или .env файл) и добавьте токен Вашего бота. Например:

# settings.py
TELEGRAM_BOT_TOKEN = '7Zmu1&UbCyJ%'

6. Миграция базы данных
Если у Вас есть миграции, выполните их:

python manage.py migrate

7. Запуск сервера
Запустите сервер с помощью команды:

python manage.py runserver 0.0.0.0:8000

8. Настройка веб-сервера (опционально)
Если Вы хотите, чтобы Ваш проект был доступен по доменному имени или через стандартный порт (80 или 443), 
Вам нужно будет настроить веб-сервер, такой как Nginx или Apache.

9. Проверка работы
Теперь Вы можете проверить, работает ли Ваш бот, отправив сообщение в Telegram.
